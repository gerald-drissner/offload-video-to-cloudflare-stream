# Security Policy

## Supported versions

Only the latest public release is supported.

## Reporting a vulnerability

Please do not open a public GitHub issue for security vulnerabilities. Contact the maintainer privately through the email address listed on the WordPress.org profile for `gerald-drissner`, or through the repository owner’s preferred private contact channel.

The plugin stores only local WordPress metadata for Cloudflare Stream videos. It never deletes videos from Cloudflare during uninstall.
